Question:

Which following statements is correct.

- [] Commit your work only when it's done is  a good practice.
- [] A good programmer do not use pull request so often, since he does not need to update with master branch.
- [] Implement UnitTest beforehand
- [] Always using name that specify the intent
- [] Think and code ahead of the requirement (To prevent unexpected flaws)

Answer:

 - [x] Commit your work only when it's done is  a good practice.
 - [ ] A good programmer do not use pull request so often, since he does not need to update with master branch.
 - [x] Implement UnitTest beforehand
 - [x] Always using name that specify the intent
 - [ ] Think and code ahead of the requirement (To prevent unexpected flaws)
